### Preinstall

#### 🚨🚨🚨 Ensure that you have the Latest Node LTS installed

### Install Dependencies

```bash
npm i
# or
yarn
```

### 🍺Run

#### Development mode:

```bash
npm run dev
# or
yarn dev
```

#### Production mode:

```bash
npm run build:start
# or
yarn build:start
```
